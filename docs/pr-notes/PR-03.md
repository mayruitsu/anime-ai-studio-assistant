# PR #03 差分チェックからdocs/を除外

## 概要

PRの差分行数チェックにおいて、`docs/` ディレクトリ以下のファイルをカウント対象から除外するよう修正した。これによりドキュメントの追記が100行制限に影響しなくなる。

## 変更ファイル

- `.github/workflows/pr-check.yml`

## 重要なコード解説

```bash
# 変更前
git diff --shortstat origin/main...HEAD

# 変更後
git diff --shortstat origin/main...HEAD -- ':!docs/'
```

`-- ':!docs/'` は Git の「pathspec」という機能で、`!` が除外を意味する。`docs/` 以下のファイルを差分カウントから除いてくれる。

## 学習ポイント

- `git diff` に `--` 以降でファイルパターンを指定できる
- `:!パターン` で特定のパスを除外できる（Git pathspec の exclude 構文）
- GitHub Actions のステップ内は通常の bash スクリプトとして書ける
