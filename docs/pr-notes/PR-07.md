# PR #07 Canvas上にモック関節ポイントを表示

## 概要

アップロードした画像をCanvasに描画し、モックの関節ポイントを赤い点で表示する機能を実装した。座標はMediaPipe接続前の仮データで、0〜1の相対値で定義している。

## 変更ファイル

| ファイル | 内容 |
|---|---|
| `frontend/src/components/PoseCanvas.jsx` | Canvas描画・ポイント表示コンポーネント（新規） |
| `frontend/src/components/ImageUploader.jsx` | プレビュー表示を削除（PoseCanvasに移譲） |
| `frontend/src/App.jsx` | PoseCanvasを組み込み |

## 処理の流れ

1. 画像をアップロードすると `App` の `image` にURLが入る
2. `PoseCanvas` に `imageSrc` として渡される
3. `useEffect` が発火し `Image` オブジェクトで画像を読み込む
4. 読み込み完了後にCanvasに画像を描画
5. `MOCK_POINTS` をループして赤い円を描画する

## 重要なコード解説

```jsx
const canvasRef = useRef(null);
```

- `useRef` はDOM要素（ここではCanvas）を直接操作するためのHooks
- `canvasRef.current` でCanvas要素にアクセスできる

```jsx
useEffect(() => {
  ...
}, [imageSrc]);
```

- `useEffect` は「特定の値が変わったときに処理を実行する」Hooks
- `[imageSrc]` は依存配列で、`imageSrc` が変わったときだけ実行される

```jsx
ctx.arc(x * img.width, y * img.height, 6, 0, Math.PI * 2);
```

- `x * img.width` で相対座標（0〜1）を実際のピクセル座標に変換
- `arc(x, y, 半径, 開始角度, 終了角度)` で円を描く
- `Math.PI * 2` は360度（円一周分）

## 学習ポイント

- `useRef` は `useState` と違い値が変わっても再描画されない。DOM操作に使う
- `useEffect` の依存配列が空 `[]` なら初回のみ、値があればその値が変わるたびに実行される
- Canvas APIはJavaScriptで図形を描画するブラウザ標準の機能
